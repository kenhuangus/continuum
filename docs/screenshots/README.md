# Screenshots (hackathon proof)

After Continuum is **Running** on Alibaba Cloud ECS, save a Workbench screenshot here as:

```
docs/screenshots/alibaba_workbench.png
```

## What to capture

- Alibaba Cloud **Workbench** (or console) showing the ECS instance status **Running**
- Blur account IDs / personal emails if you prefer before uploading to Devpost

## What not to do

- Do **not** invent or commit a fake screenshot
- Do **not** put secrets, API keys, or `/etc/continuum.env` contents in images

Until the live ECS deploy exists, this README is the only tracked file in this folder.
