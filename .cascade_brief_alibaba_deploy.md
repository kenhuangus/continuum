- Entrypoint: `uvicorn continuum_api.main:app --host 0.0.0.0 --port 8000`
- `WEB_DIR` resolves to `apps/web` relative to `apps/api/continuum_api/main.py` (parents[2]/web`) — copy `apps/web` into the image
- Packages needed on PYTHONPATH / install: `packages/memory_core`, `packages/agent`, `packages/mcp_server`, `apps/api`

## Create these files

### 1. `Dockerfile` (repo root)

```
# Build from repo root
# python:3.12-slim
# WORKDIR /app
# COPY pyproject.toml README.md ./
# COPY packages/ packages/
# COPY apps/ apps/
# pip install --no-cache-dir -e .
# ENV CONTINUUM_DB_PATH=/app/data/continuum.db
# EXPOSE 8000
# HEALTHCHECK curl or python httpx to /v1/health
# CMD uvicorn continuum_api.main:app --host 0.0.0.0 --port 8000
```
